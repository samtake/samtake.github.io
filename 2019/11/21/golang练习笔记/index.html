  • go语言只有值传递一种方式
    var a int 当参数传递给fun f(pa *int)它其实就是将a的指针&a拷贝一份传递给pa,pa也跟&a一样，同时指向a

  • 再如：var cache Cache 当参数传递给func f(cache Cache),参数Cache那么大不可能拷贝一份的，实际上Cache也只是一个指向data的指针，所以cache当参数的时候，它只是拷贝了一份指针，它同时也指向了data
